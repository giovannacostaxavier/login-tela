# 🔐 Tela de Login Responsiva

Projeto de uma **Tela de Login moderna e responsiva**, desenvolvido utilizando apenas **HTML e CSS**.

Este projeto foi criado com o objetivo de praticar estruturação de layout, estilização avançada e responsividade com Media Queries.

---

## 🚀 Demonstração

🔗 Acesse o projeto online:  
👉 https://giovannacostaxavier.github.io/login-tela/

---

## 📌 Funcionalidades

- Estrutura semântica em HTML5  
- Estilização moderna com CSS3  
- Layout organizado e centralizado  
- Design responsivo  
- Adaptação para diferentes tamanhos de tela usando **Media Queries**

---

## 🛠 Tecnologias Utilizadas

- HTML5  
- CSS3  
- Media Queries  
- Visual Studio Code  

---

## 📂 Estrutura do Projeto

- `index.html` – Estrutura da página  
- `style.css` – Estilização principal  
- `media-query.css` – Responsividade  
- `img/` – Imagens utilizadas no projeto  

---

## 📱 Responsividade

O projeto foi desenvolvido pensando na adaptação para diferentes dispositivos, garantindo uma boa experiência tanto em telas maiores quanto em dispositivos móveis.

---

## ▶ Como executar localmente

1. Clone ou baixe o repositório  
2. Abra o projeto no **VS Code**  
3. Clique com o botão direito no arquivo `index.html`  
4. Selecione **"Open with Live Server"**  

Ou abra o arquivo `index.html` diretamente no navegador.
